## 关于我

我是一名计算机科学与技术本科生，目前主要关注 **AI 应用开发、知识图谱、智能体、AI Skill 与可编辑内容生成**。

相比于单纯调用模型，我更感兴趣的是：

- 如何把复杂知识组织成可检索、可推理的结构
- 如何让 AI 生成的 PPT 真正可编辑、可复用
- 如何把大模型能力封装进具体任务流程
- 如何构建能落地到真实场景的 AI 工具
